What is CSS?

  1. CSS stands for cascading Style Sheets

  2. CSS discribe how HTML elements are to be displayed on screen,paper,or in other media.

  3. Css is not neccesaryCSS saves a lot of work.It can control the layout of multiple web pages all at once

  4. External stylesheets are stored in CSS files.

CSS can be added to HTML documents in 3 ways:

  • Inline - by using the style attributes inside the HTML elements
  • Internal-by using a style element in the head section
  • The most common way to add CSS, is to keep the styles in external CSS files, However, in this tutorial we will use inline internal styles,because this is easier to demostrate.
